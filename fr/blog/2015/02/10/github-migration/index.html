Déménagement de mes extensions sur Github post

La Forge de TYPO3 est une place intéressante qui regroupe (regroupait?) dans un même endroit un ensemble d'extensions et de ressources. Malheureusement, La plate-forme a négocié avec retard le virage vers Git et bon nombre de projets ont migré vers Github, Bitbucket et consorts. A cela s'ajoute la difficulté de Gerrit qui est l'outil de revue qui est assez touffu à prendre en main.

Afin de faciliter les contributions sur mes projets, j'ai décidé de déménager mes extensions sur Github pour des raisons pragmatiques.

  • context_hints
  • media
  • vidi
  • vidi_starter
  • vidi_frontend
  • quick_form
  • messenger
  • rss_display
  • metadata
  • media_upload
